Sikkim Tourism & Cultural Preservation Platform (Frontend)

👉 https://m360s.netlify.app

--> Overview

This project is a modern web platform built to promote tourism in Sikkim while preserving its cultural heritage. It provides an immersive user experience with virtual exploration, trip planning, and controlled access to sensitive historical manuscripts.

--> Features
🌍 Tourism Exploration
Browse popular destinations in Sikkim
Categorized places (monasteries, lakes, mountains, etc.)
Rich UI with images and details
🕶️ VR Experience
Virtual Reality view of tourist spots
Interactive exploration for users
🎫 Trip Booking UI
User-friendly booking interface
Select destinations and plan trips
Clean and responsive design
📜 Manuscript Access (UI Concept)
Separate views for:
General public
Local community
Restricted cultural data
Designed for secure access (backend integration pending)
🛠️ Tech Stack
Frontend: React.js
Styling: CSS / Tailwind (if used)
Deployment: Netlify
